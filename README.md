## Background:
This project uses:
  * `gulp` to compile `.scss`
  * `normalize.css` to standardize browser inconsistences
  * `gulp-autoprefixer` to add CSS vendor prefixes

## Instructions:
* Install dependencies: `gulp` `gulp-sass` `gulp-autoprefixer` 
* Run command `gulp sass-build`
* Open `index.html`
